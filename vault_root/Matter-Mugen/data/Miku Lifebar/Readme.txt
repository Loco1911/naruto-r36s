Welcome to the Readme file of "Hatsune Miku: Project MUGEN S.P. Lifebar". Before we get started, please meet the following requirements as shown:

====================================================
- Winmugen or above
- Video needs to be set to 640x480.
- DoubleRes needs to be set to 4. NOTE: Winmugen only, Not applicable for Winmugen+.
====================================================

History:
As far as I know, there hasn't been any MUGEN related VOCALOID screenpack nor lifebar, so I decided to create one. The project started on July 2010.

Installation:
Extract the zipped file in your data folder. NOTE: If you don't want your fight files replaced, extract it as a folder instead.

After extracting, there are four def files shown on the folder: 

====================================================
- fight.def: 		Custom portraits for 1P and 2P, two-digit timer.
- fight_999.def: 	Custom portraits for 1P and 2P, three-digit timer. NOTE: Time has to be set to 999.
- fight_normal.def: 	Normal portraits for 1P and 2P, two-digit timer.
- fight_normal_999.def:	Normal portraits for 1P and 2P, three-digit timer. NOTE: Time has to be set to 999.
====================================================

Both fight.def and fight_999.def are custom portrait styles for 1P and 2P. Custom portraits have to be 9000,35. I've added a template to create custom portraits of your own.
