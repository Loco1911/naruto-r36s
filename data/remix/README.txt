=========================================< INSTALLATION >==========================================

 1 - Open your Ikemen GO's "data" folder

 2 - Create a new subfolder called "mugen1_screenpack_remix"

 3 - Move the contents of the archive into the new folder

 4 - Inside Ikemen's "save" folder, open the "config.json" file with a text editor

 5 - Look for this line:

  "Motif": "data/system.def",

 6A - For 16:9 screen aspect ratio, replace the line with this:

  "Motif": "data/mugen1_screenpack_remix/system_169.def",

 6B - For 4:3 screen aspect ratio, replace the line with this:

  "Motif": "data/mugen1_screenpack_remix/system_43.def",



=========================================< INTRODUCTION >==========================================

This is an enhanced edit of Ikemen GO's default high resolution screenpack.

It features a new color scheme, new layouts, menu music and some Ikemen GO features not actually found in the original.



============================================< F.A.Q. >=============================================

Q: Can I use this screenpack with Orange-Dolphin's palette select branch?
|
A: Yes, you can. It is already configured for that, so all you need to do is install the screenpack.
